# 📄 TABU.AI

![TABU.AI Logo](https://via.placeholder.com/200x80/4F46E5/white?text=TABU.AI)

## 🎯 Vizyon
**"Yasaklı kelimeler, sınırsız yaratıcılık!"**

TABU.AI, kullanıcıların yapay zeka ile etkili iletişim kurma becerilerini geliştirmesini sağlayan, oyunlaştırılmış bir prompt engineering platformudur. Klasik TABU oyunundan ilham alan bu uygulama, hem öğretici hem de eğlenceli bir deneyim sunar.

## 🎮 Nasıl Çalışır?

1. **Kelime Gösterimi**: Hedef kelime ve tabu kelimeler gösterilir
2. **Prompt Yazma**: Tabu kelimeleri kullanmadan açıklayıcı prompt yazın
3. **AI Değerlendirmesi**: Yapay zeka tahminini yapar
4. **Sonuç**: Doğru tahmin = Başarı! 🎉

## 🚀 Özellikler

### 🎯 Oyun Modları
- **Tekli Oyun**: Zaman sınırlı pratik

## 🛠️ Teknoloji Stack

- **Frontend**: Angular + TypeScript
- **Backend**: .NET 9 (Clean Architecture)
- **Database**: PostgreSQL
- **AI**: OpenAI API
- **Containerization**: Docker

## 📦 Kurulum

### Gereksinimler
- .NET 9 SDK
- Node.js 18+
- PostgreSQL
- Docker (opsiyonel)

### Hızlı Başlangıç

```bash
# Repository'yi klonlayın
git clone https://github.com/username/tabu-ai.git
cd tabu-ai

# Backend'i çalıştırın
cd backend
dotnet restore
dotnet run

# Frontend'i çalıştırın
cd ../frontend
npm install
ng serve
```
